1. 快速开始

创建一个 .toml 文件，填写需要自定义的字段，然后在页面底部点击「导入主题」按钮选择该文件即可。未填写的字段自动使用默认值。

也可以直接使用页面底部的预设主题下拉菜单，从 7 个内置主题中一键切换。

# my-theme.toml — 最简示例
name = "我的主题"

[light]
page_bg = "#f0f4f8"
heading_color = "#1a365d"
tab_active_color = "#3182ce"
tab_active_border = "#3182ce"
timeslot_color = "#3182ce"
link_color = "#3182ce"
提示：项目 assets/themes/ 目录下提供了 7 个预制主题（Tokyo Night、Nord、Catppuccin、Gruvbox、Ayu、Everforest、Kanagawa），可通过下拉菜单直接选择，也可下载 TOML 文件作为编写自定义主题的参考。

2. 命名规则

所有配置项遵循统一的命名格式：

元素名_CSS属性

元素名标识页面中的 UI 组件，CSS属性标识要控制的样式属性。两者用下划线连接。

元素名一览

元素名对应 UI
page页面整体（<body>）
heading主标题（<h1>）
date日期显示文字
banner顶部横幅通知栏
badge更新状态徽章
containerTab 容器 / 折叠面板容器
infobox紧急信息 / 格言通知框
info信息文本（更新时间、版本号等）
tabbarTab 按钮栏
tabTab 按钮（工学馆/本部其它/南校区）
date_btn日期切换箭头按钮
timeslot时段标题（1-2节、3-4节…）
table数据表格
floor楼层标签（1F、2F…工学馆左侧）
building教学楼名称行（本部其它/南校区）
collapsible折叠面板标题（使用须知/Credits）
collapsible_desc折叠面板内部描述文字
theme_btn主题控制按钮（导入/重置/下拉）
theme_name当前主题名称显示
hr分割线
link超链接
data数据单元格（教室数据，含普通/加粗/删除线/下划线状态）

CSS 属性后缀

后缀含义示例值
bg背景色"#f0f4f8"
color文字色"#333333"
border边框色"#d0d0d0"
shadow阴影"0 2px 8px rgba(0,0,0,0.1)"
font_size字号（仅布局）"14px"
decoration_color装饰线颜色（下划线/删除线）"#c62828"
decoration_style装饰线样式"solid" / "dashed" / "dotted" / "wavy"
decoration_thickness装饰线粗细"1.25px"
示例：banner_bg = 横幅(banner) + 背景色(bg)；tab_hover_color = Tab按钮(tab) + 悬停状态(hover) + 文字色(color)。

3. 明暗模式锁定

默认情况下，页面自动跟随系统明暗模式。可通过 mode 字段锁定为某一模式：

mode = "dark"  # 锁定暗色模式
# mode = "light"  # 锁定亮色模式
# 省略则跟随系统
注意：锁定模式后，即使用户系统切换了明暗模式，页面也会保持锁定的模式。

4. 导入与管理

  • 导入主题 — 点击页面底部的「导入主题」按钮，选择 .toml 文件
  • 预设主题 — 使用下拉菜单从 7 个内置主题中选择
  • 重置主题 — 点击「重置」按钮恢复默认样式
  • 持久化 — 主题配置保存在浏览器 localStorage 中，刷新页面后自动恢复
提示：预设主题和自定义导入互斥——选择预设会清除自定义配置，导入自定义会清除预设选择。

9. 常见问题

Q: 导入后没有变化？

检查 TOML 语法是否正确——引号遗漏、段落名拼写错误都会导致解析失败。可在浏览器控制台查看是否有报错。

Q: 只需要修改一两个变量？

完全可以。只需在 TOML 中写需要覆盖的字段，其余自动使用默认值。

Q: 如何让暗色模式和亮色模式使用不同颜色？

分别在 [light] 和 [dark] 段落中配置。省略的字段使用默认值。

Q: 主题配置保存在哪里？

保存在浏览器的 localStorage 中，有效期一年。清除浏览器数据会丢失自定义主题。

Q: 能否修改页面整体宽度？

不能。页面宽度固定为 800px，不在主题配置范围内。
